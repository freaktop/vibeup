# Missing Features in VibeUp MVP

## Current Features ✅
- Discover feed with profile cards
- Like/Save functionality
- Matches screen
- Messages list (placeholder)
- Profile editing
- Local storage (offline-first)

## Missing Features (Priority Order)

### High Priority
1. **Swipe Gestures** - Left swipe (pass) and right swipe (like)
2. **Pass/Reject** - Currently only has Like, need Pass button
3. **Multiple Photos** - Profile cards only show one photo, need photo gallery
4. **Real Chat** - Messages screen is just a list, need actual chat threads
5. **Filters** - Age range, distance, interests filtering

### Medium Priority
6. **Onboarding** - Welcome screens, tutorial
7. **Photo Upload** - Users can't upload their own photos yet
8. **Match Animation** - Celebration when you get a match
9. **Super Like** - Special like feature
10. **Undo** - Undo last swipe

### Low Priority (Nice to Have)
11. **Location Services** - Real distance calculation
12. **Push Notifications** - Match/message notifications
13. **Video Profiles** - Short video intros
14. **Verified Badges** - Profile verification
15. **Block/Unmatch** - Remove matches
16. **Share Profile** - Share profiles with friends
17. **Discovery Settings** - Control who sees you
18. **Read Receipts** - See if messages are read

## Quick Wins (Easy to Add)
- Add Pass button to Discover
- Add photo gallery to ProfileCard
- Add filters dropdown
- Add match celebration animation
- Add undo button
